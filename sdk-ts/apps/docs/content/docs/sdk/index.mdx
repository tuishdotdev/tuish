# Tuish SDK

The `@tuish/sdk` package provides a simple API for adding licensing and payments to your terminal applications.

## Installation

```bash
npm install @tuish/sdk
```

## Quick Start

```typescript
import { Tuish } from '@tuish/sdk';

const tuish = new Tuish({
  productId: 'prod_xxx',
  publicKey: 'MCowBQYDK2VwAyEA...',  // Ed25519 public key
});

// Check license
const result = await tuish.checkLicense();

if (result.valid) {
  console.log('Licensed!', result.license);
} else {
  // Open browser for purchase
  const session = await tuish.purchaseInBrowser();
  const license = await tuish.waitForCheckoutComplete(session.sessionId);
}
```

## Configuration

```typescript
interface TuishConfig {
  /** Product ID for this application */
  productId: string;

  /** Ed25519 public key for offline license verification */
  publicKey: string;

  /** API base URL (defaults to production) */
  apiBaseUrl?: string;

  /** API key for authenticated requests */
  apiKey?: string;

  /** Custom storage directory (defaults to ~/.tuish/licenses/) */
  storageDir?: string;

  /** Enable debug logging */
  debug?: boolean;
}
```
