# Tuish Python SDK

The `tuish` package provides licensing, checkout, and offline verification for
Python-based CLI apps.

## Installation

```bash
pip install tuish
```

## Quick Start

```python
from tuish import Tuish

client = Tuish(
    product_id="prod_xxx",
    public_key="MCowBQYDK2VwAyEA...",
)

result = client.check_license()

if result.valid:
    print("Licensed!")
else:
    session = client.purchase_in_browser()
    client.wait_for_checkout_complete(session.session_id)
```

## Configuration

```python
Tuish(
    product_id="prod_xxx",
    public_key="MCowBQYDK2VwAyEA...",
    api_base_url="https://tuish-api-production.doug-lance.workers.dev",
    api_key="tuish_sk_...",
    storage_dir="~/.tuish/licenses",
    debug=False,
)
```

## Notes

- License verification is offline-first and uses Ed25519 signatures.
- Cached licenses are stored under `~/.tuish/licenses/` by default.
- Async support is available via the underlying client APIs.
