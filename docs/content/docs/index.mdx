# Getting Started with Tuish

**Tuish is "better-auth for entitlements"** — drop-in licensing and monetization for terminal apps.

```
better-auth              Tuish
───────────────          ───────────────
Auth primitives      →   Entitlement primitives
Session tokens       →   License keys
Web components       →   TUI components (Ink)
Login/signup UI      →   Purchase/manage UI
```

In terminal environments, **the license key IS the auth**. We don't do authentication — we do entitlements.

## Quick Start

### Option 1: Drop-in TUI Components (Recommended)

```bash
npm install @tuish/ink
```

```tsx
import { render } from 'ink'
import { LicenseProvider, LicenseGate, PurchaseFlow } from '@tuish/ink'

function MyCliApp() {
  return (
    <LicenseProvider productId="your-product-id">
      <LicenseGate
        feature="pro"
        fallback={<PurchaseFlow />}
      >
        <ProFeature />
      </LicenseGate>
    </LicenseProvider>
  )
}

render(<MyCliApp />)
```

### Option 2: SDK Only

```bash
npm install @tuish/sdk
```

```typescript
import { Tuish } from '@tuish/sdk'

const tuish = new Tuish({
  productId: 'your-product-id',
  publicKey: 'your-public-key',
})

const result = await tuish.checkLicense()

if (result.valid) {
  runYourApp()
} else {
  await tuish.purchaseInBrowser()
}
```

The SDK handles checkout, license storage, offline verification, and cache refresh.

## TUI Components

Embeddable terminal components for license management:

| Component | Purpose |
|-----------|---------|
| `<LicenseProvider />` | Context provider for license state |
| `<LicenseGate />` | Gate features, show upgrade prompt |
| `<PurchaseFlow />` | QR code + URL checkout in terminal |
| `<LicenseStatus />` | Show current plan |
| `<TuishLicenseManager />` | Complete self-service UI |

## Key Features

| Feature | Description |
|---------|-------------|
| **Drop-in components** | Embed purchase flow and license management in your TUI |
| **Offline-first** | Licenses work without internet after activation |
| **Cryptographic verification** | Ed25519 signed, tamper-proof licenses |
| **Your Stripe account** | Payments go directly to you (Standard Connect) |
| **Zero liability** | You handle billing in Stripe, we handle licenses |
